## Gold Rush
> Unity Version: 2022.3.5f1

A game similar to  Subway Surfers, Minion Rush: Running, Temple Run 

## Additional Packages Used
- DOTween
- Epic Toon FX
- GUI Packages
- Platformer Game Kit

## Mechanics Made
- Player Movement
- Player Ragdoll
- Obstacle Collision
- Gold Jump Trajectory Instantiator
- Boosters

    >
    > Double Gold Booster
    >
    > Magnet Booster
    >
    > Jump Boost Booster
    >
- Custom Water Shader Graph
